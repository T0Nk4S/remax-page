A continuación se presenta el modelo ambiental del Sistema de Bienes Raices.

Declaración de propósitos

El Sistema Web de Gestión Inmobiliaria es una plataforma digital diseñada para resolver los desafíos del mercado inmobiliario, conectando a usuarios,clientes, agentes y proyectos en un entorno centralizado. El sistema permitirá a los usuarios consultar propiedades,a los clientes buscar propiedades según sus preferencias, a los agentes gestionar sus propiedades y responder consultas de manera eficiente, y a los proyectos mantener un control actualizado sobre su progreso y estado financiero. A través de flujos de datos claros y procesos integrados, el sistema busca reducir la duplicación de esfuerzos, mejorar la comunicación y garantizar la transparencia.

Lista de acontecimientos

1. Un usuario o cliente inicia una busqueda proporcionando criterios especificos para una o mas propiedades

2. Un usuario o cliente puede soliciar informacion de la propiedad al sistema

3. Registro de usuario al sistema web de gestion inmobiliaria convertiendose en cliente

4. Un cliente puede realizar inverciones en proyectos

6. Un cliente puede actualizar su informacion personal

7. Un agente inmobiliario registra o agrega nuevas propiedades

8. El agente inmobiliario actualiza informacion de propiedad

9. El agente revisa consultas de clientes sobre las propiedades

10. Registro de nuevos proyectos inmobiliario

11. Actualizacion del proyecto

12. Generar reportes
