Modelo de Objetos de Negocio

El Modelo de Objetos de Negocio es una representación conceptual de los elementos clave dentro de una organización o sistema y las relaciones entre ellos. Este modelo se utiliza en la Ingeniería de Software y en la Gestión de Negocios para entender y estructurar los procesos, actores y recursos necesarios para cumplir con los objetivos de negocio.
